// Estrutura repetitiva

Looping

while, enquanto, uma estrutura de controle que repete comandos enquanto uma condição for verdadeira

motivo de usar: quando não se sabe quantas repetições vão ser usadas, existem varias

parece o if

v: executa o bloco e volta no começo do bloco

f: pula o bloco e continua o algoritmo

teste de mesa

for, para, uma estrutura de contro que repete comandos um determinado numero de vezes

motivo de usar: quando se sabe quantas repetição vão ser usadas, um controle de repetições pre determinado

for (inicio; condição; incremento_ou_decremento)

inicio: executa só uma vez

condição:

v: executa e volta

f: para e continua o fluxo do algoritmo

incremento ou decremento: executa toda vez depois de uma repetição

for é ótimo para contagens, melhor que while nesse caso

estrutura de controle do-while, faça-enquanto

que é uma estrutura um pouco utilizada, rara

mas para alguns problemas se encaixa melhor

o comando é executado pelo menos uma vez, porque a condição só é verificada no final

v: volta e repete

f: só continua o fluxo
